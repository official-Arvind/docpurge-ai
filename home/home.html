// HOW IT WORKS

Five phases. One perfect output.

🔍
Phase 1 — Auto-Detection
pdf-lib parses every content stream, identifies Form XObjects, transparency layers, and repeated text stamps across all pages.
🗑️
Phase 2 — Native Removal
Detected watermark objects are deleted natively from the PDF structure. No images. No re-encoding. Vectors and fonts stay 100% intact.
💬
Phase 3 — Hint UI
If auto-detection draws a blank, a clean input panel slides in. You type a hint like "DRAFT" or "Confidential" and we search specifically for it.
🤖
Phase 4 — Gemini AI Assist
Page text is extracted with pdf.js and sent to Gemini 3.5 Flash via your own API key. Gemini returns the exact watermark string. Your key never leaves JS memory.
📥
Phase 5 — Export
The Gemini-identified string is matched across all pages and surgically removed. The clean PDF is assembled and immediately downloaded — never stored.
// FAQ

Straight answers. No fluff.

Does my PDF get uploaded anywhere?
Absolutely not. DocPurge AI runs entirely in your browser. The PDF is loaded into browser memory (ArrayBuffer), processed with pdf-lib and pdf.js, and the output is downloaded directly from that memory. Zero network requests are made for your file. The only external call is the optional Gemini API — and that only sends extracted text (not the raw PDF), using your own key.
What's the maximum file size?
There is no enforced size limit. We've designed the engine to handle large PDFs (200MB+) using Web Workers to keep processing off the main thread. Performance will depend on your device's RAM and CPU — modern machines handle 200MB PDFs comfortably. We recommend Chrome or Edge for best performance on very large files.
Will removing a watermark damage the PDF's text or images?
No. DocPurge AI never converts pages to raster images — that would destroy text searchability and blow up file size. We operate directly on PDF content stream operators (Tj, TJ, Do), deleting only the watermark objects while leaving all other content untouched. DPI, vector sharpness, and font embedding are preserved exactly.
Is my Gemini API Key safe?
Yes. Your key is stored in a JavaScript variable within the current browser session only — it is never written to localStorage, IndexedDB, cookies, or any external server. When you close or refresh the tab, the key is gone. The Gemini API call goes directly from your browser to Google's servers, bypassing DocPurge AI's infrastructure entirely (which doesn't exist).
What types of watermarks can it remove?
Phase 1 targets: PDF Form XObjects used as overlays, text stamps with transparency (common in Adobe-stamped PDFs), repeated text patterns across pages (e.g., "DRAFT", "CONFIDENTIAL"), and image watermarks embedded as separate XObjects. Complex watermarks baked directly into the content stream may require Gemini AI Assist (Phase 4).
Which Gemini models are available?
We support the current 2026 Gemini lineup: Gemini 3.5 Flash (fastest, recommended for most watermarks), Gemini 3.1 Pro (most accurate, best for complex/obfuscated watermarks), and Gemini 3.1 Flash-Lite (cheapest, good for simple text hints). You can select your preferred model in the tool interface.
